WiFi密码查看：已连接WiFi密码如何找回

2026-06-09  |  电脑技巧

方法1：Win+R → 输入ncpa.cpl → 右键WiFi → 无线属性 → 安全 → 勾选'显示字符'。

方法2：Win+X → 终端管理员 → 输入 netsh wlan show profiles → 找到WiFi名称 → netsh wlan export profile name='WiFi名称' folder=C:\ → 打开生成的XML文件找到关键内容。

方法3：下载WirelessKeyView（小工具），直接显示本机所有保存的WiFi密码。
